## Hey Linda

A free forever meditation app built with React Native.
### Features

- meditate
- track progress
- dark mode
- more than 100+ meditations
- current streak
- calendar activity
- time listened

## Install

To install the project, navigate to the directory and run:

- `yarn install`

## Run

To run the project, run the following commands:

- `yarn android`
- `yarn ios`
